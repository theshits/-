# 大气污染扩散模拟系统

基于高斯烟羽模型的大气污染扩散模拟系统，支持点源、面源、线源和等效面源的扩散计算。

## 功能特性

- **排放源管理**：支持点源、面源、线源、等效面源四种排放源类型
- **气象条件管理**：支持风向、风速、大气稳定度、温度、湿度、云量、降水等气象参数配置
- **受体点管理**：支持添加多个受体点，并设置高度参数
- **扩散模拟**：
  - 单一风向模拟
  - 全局模拟（支持8/16/32/64风向加权组合）
  - 风向模拟（特定风向风速组合）
- **结果展示**：
  - 浓度场热力图
  - 受体点浓度贡献分析
  - 污染物浓度分布
- **污染物支持**：PM2.5、PM10、TSP、VOCs、NOx、SO2、CO、O3

## 技术架构

- **后端**：FastAPI + SQLAlchemy
- **前端**：HTML + JavaScript + Leaflet地图
- **核心算法**：高斯烟羽扩散模型 + WRF沉降模型

## 快速开始

### 环境要求

- Python 3.10+
- pip

### 安装步骤

1. **克隆项目**
```bash
git clone <项目地址>
cd gnn
```

2. **安装依赖**
```bash
cd backend
pip install -r requirements.txt
```

3. **启动服务器**

双击运行 `start_server.bat` 或在终端执行：
```bash
cd backend
python -m uvicorn main:app --host 0.0.0.0 --port 8080 --reload
```

4. **访问系统**

打开浏览器访问：http://localhost:8080

### 关闭服务器

双击运行 `stop_server.bat` 或在运行服务器的终端按 `Ctrl + C`

## 使用说明

### 1. 排放源管理

- 支持新增、编辑、删除排放源
- 支持批量导入/导出排放源
- 支持四种排放源类型：
  - **点源**：烟囱等点状排放源，需要设置排放速率、烟囱高度、烟气温度、出口速度等参数
  - **面源**：厂房、堆场等面状排放源，需要设置面源尺寸和排放速率
  - **线源**：道路等线状排放源，需要设置起点终点坐标和排放速率
  - **等效面源**：已知浓度的监测点，系统自动计算等效排放速率

### 2. 气象条件

- **风向**（0-360度）：气象风向，风来自的方向
- **风速**（m/s）：影响扩散稀释和污染物传输
- **大气稳定度**（A-F）：
  - A：极不稳定，扩散最快
  - B：不稳定
  - C：弱不稳定
  - D：中性
  - E：弱稳定
  - F：稳定，扩散最慢
- **温度**（K）：影响化学转化速率
- **湿度**（%）：影响干沉降和化学转化
- **云量**（0-10）：影响湿沉降和化学转化
- **降水**（mm/h）：影响湿沉降
- **边界层高度**（m）：影响垂直扩散和沉降

### 3. 受体点管理

- 添加监测受体点
- 设置受体点位置和高度
- 查看各排放源对受体点的浓度贡献

### 4. 扩散模拟

- **单一风向模拟**：选择特定气象条件进行模拟
- **全局模拟**：设置多个风向风速组合及权重，进行加权模拟
- **风向模拟**：特定风向风速组合的模拟

## 高斯扩散模型

### 基本公式

系统采用高斯烟羽模型进行污染物扩散计算：

$$C(x,y,z) = \frac{Q}{2\pi u \sigma_y \sigma_z} \exp\left(-\frac{y^2}{2\sigma_y^2}\right) \left[\exp\left(-\frac{(z-H)^2}{2\sigma_z^2}\right) + \exp\left(-\frac{(z+H)^2}{2\sigma_z^2}\right)\right]$$

其中：
- C：浓度 (μg/m³)
- Q：排放速率
- u：风速
- σy, σz：水平和垂直扩散参数
- H：有效源高度

### 扩散参数（Pasquill-Gifford）

$$\sigma_y = a_y \cdot x^{b_y}$$
$$\sigma_z = a_z \cdot x^{b_z}$$

| 稳定度 | ay | by | az | bz | 扩散特点 |
|--------|-----|-----|-----|-----|----------|
| A | 0.527 | 0.865 | 0.28 | 0.90 | 极不稳定，扩散最快 |
| B | 0.371 | 0.866 | 0.23 | 0.85 | 不稳定 |
| C | 0.209 | 0.897 | 0.22 | 0.80 | 弱不稳定 |
| D | 0.128 | 0.905 | 0.20 | 0.76 | 中性 |
| E | 0.098 | 0.902 | 0.15 | 0.73 | 弱稳定 |
| F | 0.065 | 0.902 | 0.12 | 0.67 | 稳定，扩散最慢 |

### 面源处理（虚拟点源法）

对于面源，采用虚拟点源法：

1. **初始扩散参数**：
   - σy0 = W / 4.3
   - σz0 = H / 2.15

2. **虚拟距离**：
   - x_virtual = max((σy0/ay)^(1/by), (σz0/az)^(1/bz))

3. **有效距离**：
   - x_eff = x_rotated + x_virtual

4. **有效扩散参数（卷积形式）**：
   - σy_eff = √(σy² + σy0²)
   - σz_eff = √(σz² + σz0²)

### 等效面源处理

对于已知浓度的等效面源：

1. **排放速率转化**：
   - Q = C × U × H × √(L × W)
   - C：测量浓度 (μg/m³)
   - U：风速
   - H：面源高度
   - L, W：面源尺寸

2. **浓度上限**：
   - 在面源内部：C_final = min(C_calculated, C_measured)
   - 在面源外部：使用高斯扩散计算

## 衰减模型

### 1. 干沉降

采用WRF模型的阻力方法：

$$v_d = v_g + \frac{1}{R_a + R_b + R_c}$$

其中：
- vg：重力沉降速度
- Ra：空气动力学阻力（受稳定度影响）
- Rb：层流底层阻力
- Rc：冠层阻力

**稳定度影响**：
- F稳定度 → Ra最大 → 干沉降最弱
- A稳定度 → Ra最小 → 干沉降最强

**污染物参数**：

| 污染物 | vg (m/s) | Rb (s/m) | Rc (s/m) |
|--------|----------|----------|----------|
| PM2.5 | 0.0002 | 100 | 200 |
| PM10 | 0.018 | 50 | 100 |
| TSP | 0.05 | 30 | 80 |
| VOCs | 0.0 | 200 | 800 |
| NOx | 0.0 | 150 | 500 |
| O3 | 0.0 | 150 | 600 |

### 2. 湿沉降

$$\Lambda = a \cdot P^b + \Lambda_{background}$$

其中：
- P：降水强度
- a, b：污染物清除参数
- Λbackground = 1e-5（背景清除，考虑云、雾、湿空气）

**云量修正**：
- Λ × (1 + 0.1 × cloud_cover)

**污染物参数**：

| 污染物 | a | b |
|--------|-----|-----|
| PM2.5 | 1e-5 | 0.8 |
| PM10 | 2e-5 | 0.8 |
| TSP | 3e-5 | 0.8 |
| VOCs | 1e-6 | 0.7 |
| NOx | 5e-6 | 0.7 |
| O3 | 5e-6 | 0.7 |

### 3. 化学转化

$$k_{effective} = k_{base} \cdot f_T \cdot f_H \cdot f_C$$

其中：
- k_base：基础化学转化速率
- f_T：温度因子 = exp((T-298)/20)
- f_H：湿度因子 = 1 + (H-50)/200
- f_C：云量因子 = 1 + cloud_cover/50

**污染物化学转化速率**：

| 污染物 | k_base (1/s) |
|--------|--------------|
| PM2.5 | 2e-5 |
| PM10 | 1e-5 |
| TSP | 5e-6 |
| VOCs | 3e-4 |
| NOx | 1.5e-4 |
| O3 | 1e-4 |

**气体污染物增强**：
- VOCs, NOx, O3：温度因子 × 1.5，湿度因子 × 1.3

### 4. 总衰减

$$decay_{total} = decay_{deposition} \times decay_{chemical}$$

其中：
- decay_deposition = exp(-(k_dry + Λ) × t)
- decay_chemical = exp(-k_effective × t)
- t = x_eff / wind_speed

## 气象参数影响总结

| 参数 | 影响的计算 | 影响方式 |
|------|------------|----------|
| **稳定度** | 扩散参数、干沉降阻力 | A→F: 扩散减慢、沉降减弱、浓度升高 |
| **风速** | Q转化、扩散稀释、衰减时间 | 风速↑ → 浓度↓、衰减时间↓ |
| **温度** | 干沉降、化学转化 | 温度↑ → 化学转化加快 |
| **湿度** | 干沉降、化学转化 | 湿度↑ → 沉降增强、化学转化加快 |
| **云量** | 湿沉降、化学转化 | 云量↑ → 湿沉降增强 |
| **降水** | 湿沉降 | 降水↑ → 湿沉降增强 |
| **边界层高度** | 垂直扩散限制、干沉降 | BLH↑ → 沉降减弱 |
