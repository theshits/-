## 空气污染扩散模拟平台实现计划

### 第一阶段：后端基础架构
1. 创建 FastAPI 后端框架
2. 设计 SQLite 数据库模型（排放源、受体点、气象场）
3. 实现 CRUD API 接口

### 第二阶段：扩散模型核心
4. 实现高斯烟羽扩散模型
5. 实现扩散参数计算（基于Pasquill-Gifford稳定度分类）
6. 实现贡献度计算算法

### 第三阶段：前端界面
7. 创建 Vue 3 + TypeScript 前端项目
8. 实现排放源管理页面
9. 实现受体点管理页面
10. 实现气象场管理页面（含雷达状可拖动风向控件）
11. 实现主界面地图展示（Leaflet）
12. 实现扩散可视化（等值线/热力图）
13. 实现贡献度排名图表

### 第四阶段：集成与优化
14. 前后端联调
15. 界面美化（苹果风格）
16. 性能优化

**技术栈**：FastAPI + Vue 3 + Leaflet + ECharts + SQLite
